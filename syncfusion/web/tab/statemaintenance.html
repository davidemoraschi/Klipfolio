Essential Studio for ASP.NET contains all the controls you need to build line-of-business web applications—including grids, charts, gauges, menus, calendars, editors, and much more. It also comes with high-performance libraries that enable your applications to read and write Excel, Word, and PDF documents.
Our very first natively written MVC control suite contains all the controls required to put together enterprise grade web applications including grids, charts, gauges, menus, calendars, editors, and much more. It also comes with high-performance libraries that enable your applications to read and write Excel, Word, and PDF documents. You will find several one of a kind MVC controls including OlapClient, PDF Viewer, Report Viewer, etc.
Essential Studio for Mobile MVC contains all the controls you need to build line-of-business mobile-web applications including grids, charts, gauges, menus, calendars, editors, and much more. Mobile-web is a great way to build websites as well as native device apps with our Orubase offering.
